This generates a spreadsheet chart that gives the values of of a specified source for all buildings on a monthly basis. Clicking a cell in the table allows for a chart to be generated to further examine the data. If a cell is orange colored, that means there was a recorded decrease over that month (i.e. If a cell for April is orange, then that means the value decreased from March to April). The value on the first of each month is shown in the cell. Clicking 'All' will allow for graphing over entire timespan. The "reset filter" button allows for a correction of data (cumulative) when there are erronous resets in the data gathering process.

Legend:

  • This color indicates an abnormality from previous month
  • This color indicates abnormality occurs during this month (click to show on graph)
  • This color indicates Null values
  • This color indicates Nonphyiscal values
  • This color in the Account box indicates that the filtering was not completed correctly. This can occur when the filter is applied to errors beyond its scope or the data has too many gaps
